Domain 1

My 1st Domain

The domain focuses on HTML components that are associated with basic syntax, metadata, scripting, and linking. Noscript tags show content in the event that scripting is not supported, while script tags include non-HTML code—typically JavaScript—for webpage functionality. HTML files can be linked to other resources such as stylesheets using link tags, and character types, webpage content, and translation information can be defined using meta tags. The DOCTYPE, HTML, head, and body sections—each with a distinct purpose—make up an HTML document. Website annotations and extra functionality are offered via HTML entities and comments. Project completion in this area allows you to become proficient with script, noscript, link, and meta tags. It also helps you learn HTML entities, syntax, and comments so that you can create webpages that work.

Domain 2

The principles of Cascading Style Sheets (CSS) and how they are used in web design are the core topics of this area. It describes the order of styles and covers the construction and use of style sheets with external, internal, and inline styles. Students gain an understanding of CSS rule sets, syntax, and how classes, IDs, and pseudo classes are applied to HTML components via hands-on projects. Through the completion of several activities, students become proficient in the creation and use of CSS styles to improve webpage formatting and appearance, which eventually enhances their comprehension of web development principles.

My 2nd Domain

Domain 3

My 3rd Domain

The domain includes a wide range of HTML topics, including form components, accessibility features, and fundamental structure. Starting with the basics of HTML—tables, headers, paragraphs, and semantic tags—it moves on to more complex subjects like form developing, navigation, and linking. Every segment establishes distinct assignments and procedures to be fulfilled, like making tables with certain characteristics, introducing hyperlinks with objectives, combining image links, and increasing accessibility through input forms and buttons. The usage of datalist and fieldset/legend tags for form organization, as well as map and area tags for image maps, are examples of advanced subjects. The domain's overall goal is to provide students a thorough grasp of HTML components and how to use them practically in web development, with a focus on both structure and usefulness.

Domain 4

My 4th Domain

This domain covers many facets of integrating multimedia into HTML documents, such as adjusting image dimensions and linking, creating tables with text and images, embedding videos with multiple source formats and controls, integrating audio with controls and subtitle tracks, and using iframes to embed external content into webpages. Clear instructions on how to complete tasks, such adding tags, attributes, and material to accomplish particular functionality, are provided in each area. Through these projects, students improve their comprehension of the principles of web development by gaining real-world experience in successfully integrating multimedia components into HTML texts.

Domain 5

My 5th Domain

This domain explores a number of CSS-related topics and offers thorough instructions on webpage layout, positioning, styling, and text formatting. height, max-width, float placement, relative and absolute positioning, static and fixed positioning, margins, padding, display attributes, alignment settings, overflow, font families, font colors, font styles, font weights, and link colors are just a few of the subjects it covers. Learners become proficient in utilizing CSS to control the visual presentation and layout of web content, guaranteeing a consistent and interesting user experience, through hands-on projects and detailed instructions.

Domain 6

My 6th Domain

The domain addresses several facets of web development, emphasizing effective coding techniques and guaranteeing usability and accessibility across a variety of browsers and devices. It places a strong emphasis on adopting web-safe fonts for uniform presentation, testing webpage styles across many browsers, integrating comments to promote collaboration, and reusing code to save time and reduce mistakes. It also emphasizes the significance of keeping structure and style distinct, offering text alternatives for accessibility, upholding proper typography and color contrast, structuring text hierarchically using HTML tags, spotting and fixing syntax errors, managing tag mismatches, and reducing cascading problems in CSS rule sets. With the use of realistic projects and completion guidelines, the domain seeks to provide students a thorough grasp of web development concepts and methods.
